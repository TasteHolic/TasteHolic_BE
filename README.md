### 커밋 스타일

---

- feat → 기능
- fix → 버그수정, 안되는 걸 고침
- refac → 오타 수정이나 디렉토리 변경 작은 수정
- chore → 코드 수정 없는 수정 (????????) - setup
- docs → readme 관리
- deploy → 배포 관련으로
- examples:
  feat: add kakao login
  deploy: update client id
